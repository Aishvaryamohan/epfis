Electronic Primary Fuel Injection System

Simple code for controlling the pulse to the Electronic fuel injector from the reference of engine TDC Position which should be in 5 bar pressure powered by a pump for the dual fuel automobile engine.

Apparatus

ECU
  • Arduino Board
  • Laptop with arduino installed laptop
Engine
  • Dual fuel engine with proper coolant system
  • TDC Position sensor (Proximity sensor)
  • Electronic Fuel injector
  • Fuel Electrical Pump
Measurement
  • DSO (Digital Storage Oscilloscope)
  • Graduated cylinders / Volumetric flask
